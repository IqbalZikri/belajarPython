Untuk Menjalankan flask bisa dengan :
1. Langsung dari nama file pythonnya : py / python main.py
2. bisa dengan menggunakan file tambahan bernama .env untuk membuat environment flask itu sendiri (bisa di cek di .env) lalu jalankan di terminal dengan : flask run

note :
Jika ingin menggunakan .env sendiri pada python maka harus menginstall pip python-dotenv terlebih dahulu
